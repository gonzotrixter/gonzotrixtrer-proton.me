## Project: CNC Relief Pipeline

Automated photo → heightmap PNG pipeline for a 3-axis CNC milling machine. A phone photo (via Syncthing) drops into a watched folder on a Windows PC → AI depth estimation → heightmap sent via Telegram to the operator.

**Target machine:** Windows 10/11 workshop PC, non-technical operator (uncle). Install is fully automated — one double-click on `install.bat`.

**Active branch:** `claude/work-in-progress-evkDk` — always develop here, never push directly to `main`.

---

## Architecture

```
install.bat          → one-click setup for uncle's PC (downloads repo ZIP, pip install, MiDaS preload)
configure.bat        → runs setup_wizard.py (Roman runs this when he arrives)
start.bat            → launches pipeline/main.py
diagnostics.bat      → read-only system check, outputs screenshot-friendly report

setup_wizard.py      → interactive Telegram bot configurator (token + chat_id → config.yaml)
preload_models.py    → downloads MiDaS model to torch.hub cache at install time

pipeline/
  main.py            → watchdog observer on C:\CNC-Pipeline\incoming\ + threading.Queue worker
  qc.py              → OpenCV quality gate (Laplacian blur + brightness)
  segment.py         → rembg background removal → alpha_to_white JPEG
  depth.py           → MiDaS_small depth estimation (lazy-loaded singleton)
  heightmap.py       → normalize + resize + optional blur → 512×512 PNG for ArtCAM
  notifier.py        → Telegram Bot API send_message/send_photo with 3-retry backoff
  requirements.txt   → pip dependencies

config.yaml          → runtime config (generated by setup_wizard.py, never committed)
config.yaml.template → template with all tunable parameters
```

**Pipeline flow:**
```
incoming/ file → _wait_for_file() polls until stable → job_queue →
  qc.check_quality() → [rejected/ + Telegram] if bad →
  segment.remove_background() →
  depth.estimate_depth() (MiDaS_small) →
  heightmap.process_heightmap() →
  output/{timestamp}_{name}/ + Telegram photo notification
```

## Key design decisions

- **`config.yaml` path**: resolved via `os.path.dirname(os.path.dirname(os.path.abspath(__file__)))` in `notifier.py` and `heightmap.py` — never hardcoded `C:\`
- **MiDaS invert**: `255 - depth` so raised surfaces = white, matching ArtCAM convention
- **File stability**: `_wait_for_file()` polls size every 0.5s (Syncthing finishes writing before processing starts)
- **Model loading**: `depth.py` uses a module-level singleton (`_model`, `_transform`) loaded on first call
- **Logging**: `logging.basicConfig()` only inside `if __name__ == "__main__":` in `main.py` — not at module level
- **Telegram retries**: exponential backoff `2^attempt` seconds, 3 attempts, then `logging.warning` (never crashes pipeline)

---

## .bat files rule

All `.bat` files must use **plain ASCII only** in `echo` statements — no box-drawing characters (`╔ ║ ═`), no Cyrillic. Windows cmd.exe with `chcp 65001` misinterprets these as commands. Use `=` and `-` for borders.

## Config parameters (config.yaml)

| Key | Default | Description |
|-----|---------|-------------|
| `quality.min_blur_score` | 80 | Laplacian variance threshold |
| `quality.min_brightness` | 30 | Min mean pixel value |
| `quality.max_brightness` | 245 | Max mean pixel value (overexposure) |
| `heightmap.size` | [512, 512] | Output resolution for ArtCAM |
| `heightmap.blur_kernel` | 3 | Gaussian smoothing (odd int, 1 = off) |
| `heightmap.invert` | true | Invert depth (raised = white) |

## Install flow (what install.bat does)

1. Admin elevation via PowerShell `RunAs`
2. Python check → `winget install Python.Python.3.11` → PATH refresh → retry loop → `.exe` fallback
3. `mkdir C:\CNC-Pipeline\{incoming,output,rejected,processing,logs,pipeline}`
4. Download repo ZIP from `https://github.com/gonzotrixter/gonzotrixtrer-proton.me/archive/refs/heads/main.zip`
5. Extract → copy `pipeline/` + `setup_wizard.py`, `preload_models.py`, `configure.bat`, `start.bat`
6. `pip install -r requirements.txt` + PyTorch CPU (`--index-url https://download.pytorch.org/whl/cpu`)
7. `winget install SyncThing.SyncThing`
8. Run `preload_models.py` with 10-min PowerShell timeout (downloads MiDaS ~200MB)
9. Create "CNC Settings" desktop shortcut → `configure.bat`
